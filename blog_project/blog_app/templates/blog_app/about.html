About My Django Blog

Project Overview

This blog was built using Django, a high-level Python web framework. It implements modern development practices and features a clean, responsive design using Tailwind CSS.

Key Features

  • Class-Based Views for efficient code organization
  • User Authentication system complete with ability to change and reset Password
  • Responsive design with Tailwind CSS
  • CRUD operations for blog posts
  • Pagination

Technical Stack

  • Django Framework
  • Tailwind CSS
  • SQLite Database
  • Python
